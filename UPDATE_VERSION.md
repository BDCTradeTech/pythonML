# Actualizar VERSION

Antes de cada deploy, actualizar en `main.py`:

```python
VERSION = "2.YY.MM.DD.HH"
```

Formato: año 2 dígitos . mes . día . hora (24hs)

Ejemplo: 26 de mayo a las 15hs → `"2.26.05.26.15"`

---

Instrucción para Claude Code:

> "Actualizar VERSION en main.py con fecha y hora actual antes del commit"
